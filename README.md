# Ironwood Coffee — Landing Page

Landing page untuk **Ironwood Coffee**, cafe industrial specialty coffee dengan konsep bold dan no-nonsense.

## Tech Stack

- [Astro 6](https://astro.build) — SSG framework
- [Tailwind CSS v4](https://tailwindcss.com) — via `@tailwindcss/vite`
- [React 19](https://react.dev) — interactive islands (`client:load` / `client:visible`)
- [Framer Motion](https://www.framer.com/motion/) — animasi
- [Solar Icons](https://github.com/480-Design/Solar-Icon-Set) via [Iconify](https://iconify.design) — icon library

## Sections

1. Navbar — fixed, scroll-aware
2. Hero — full-bleed dark, Framer Motion reveal
3. About — origin story + stats
4. Menu Highlights — tabbed (Coffee / Non-Coffee / Food)
5. The Space — gallery grid + lightbox
6. Specialty Coffee — 3 cards
7. Work From Here — animated checklist
8. Events — Coffee Workshop, Live Acoustic, Community Table
9. Testimonial — auto-rotate carousel
10. Visit Us — map embed + info
11. Footer

## Development
